1️⃣ Loops kya hain?
Loops ek tarah ka programming tool hain jo code ko baar-baar repeat karne ke liye use hota hai.

Python main mainly 2 types ke loops hote hain:

for loop – jab hume pata ho kitni baar code chalana hai.

while loop – jab hume code tab tak chalana ho jab tak koi condition true ho.

2️⃣ For Loop
for loop ka use sequence (list, string, range) par iterate karne ke liye hota hai.

Syntax:
for variable in sequence:
    # code jo repeat hoga
Example 1: List ke elements print karna
fruits = ["apple", "banana", "mango"]

for fruit in fruits:
    print(fruit)
Output:

apple
banana
mango
Example 2: Numbers ke liye range use karna
for i in range(5):  # 0 se 4 tak
    print(i)
Output:

0
1
2
3
4
Note: range(5) 0 se start hota hai aur 5 se ek kam tak jaata hai.

3️⃣ While Loop
while loop ka use tab tak repeat karne ke liye hota hai jab tak condition True ho.

Syntax:
while condition:
    # code jo repeat hoga
Example 1: Simple counting
i = 0
while i < 5:
    print(i)
    i += 1   # i = i + 1
Output:

0
1
2
3
4
Example 2: User input se loop
password = ""
while password != "secret":
    password = input("Enter password: ")
print("Access granted!")
Yeh tab tak repeat hoga jab tak user correct password na daale.

4️⃣ Loop Control Statements
Python main kuch statements hain jo loop ko control karte hain:

break – loop ko turant rok deta hai.

continue – loop ke current iteration ko skip kar deta hai aur next iteration par chala jaata hai.

pass – placeholder, kuch nahi karta.

Example: break & continue
for i in range(1, 6):
    if i == 3:
        continue   # 3 ko skip karega
    if i == 5:
        break      # loop yahan stop hoga
    print(i)
Output:

1
2

2️⃣ continue
continue ka use current iteration ko skip karne ke liye hota hai, aur loop agla step continue karta hai.

Example:
for i in range(1, 6):
    if i == 3:
        continue   # 3 ko skip karega
    print(i)
Output:

1
2
4
5
Explanation:
Jab i 3 hota hai, continue ke wajah se print statement skip ho jaata hai, lekin loop 4 se continue hota hai.

💡 Tip:

break → loop poori tarah rok deta hai

continue → sirf current iteration skip karta hai
